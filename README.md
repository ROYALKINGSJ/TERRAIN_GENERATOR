# TERRAIN_GENERATOR

## How to use :

Clone the repository in your local workspace (your computer)

<br>

Run the bash file named "run_terrain.sh"

<br>

Enter the value of "n" which is the size of 2D plane, like how many points are there in the XY plane

<br>

A 3D graph is generated based on the the internal calculations

<br>

Terrain Generated !
